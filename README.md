# Альбион-1: Интегратор Битрикс24

Профессиональный веб-сайт компании **Альбион-1**, специализирующейся на интеграции Bitrix24, оптимизации бизнес-процессов, цифровом маркетинге и разработке голосовых роботов на основе LLM.

## О проекте

Сайт разработан для позиционирования компании как официального партнёра Bitrix24 и демонстрации экспертизы в области:

- **Интеграция Bitrix24** — полный цикл внедрения CRM с нуля, миграция данных, кастомизация
- **Оптимизация бизнес-процессов** — аудит, проектирование, система операционного учета
- **Цифровой маркетинг** — сквозная аналитика, лидогенерация, оптимизация каналов
- **Голосовые роботы на LLM** — AI-Powered BDR, виртуальные колл-центры, NLP-боты

## Технологический стек

- **Frontend:** React 19, TypeScript
- **Styling:** Tailwind CSS 4 с OKLCH цветовой палитрой
- **UI Components:** shadcn/ui
- **Routing:** Wouter
- **Build Tool:** Vite
- **Server:** Express.js

### 3. Кейсы (Case Studies)
Подробное описание 4 успешных проектов:

#### БиоПрайм (BioPrime)
- **Результат:** 350 MQL/мес (B2C), 150 MQL/мес (B2B)
- **Решение:** Интеграция Bitrix24, Planado, MySnab, 1С, система операционного учета
- **Маркетинг:** Сквозная аналитика, объединение 12 рекламных кабинетов, новые каналы (Programmatic, Avito, DMP)

#### Маринэк (Marinec)
- **Результат:** Создание отдела маркетинга с нуля, запуск маркетплейсов
- **Решение:** CRM-система, интернет-магазин на Bitrix CMS
- **Маркетинг:** B2B маркетплейсы, контентная поддержка

#### СтолБери (Stolberi)
- **Результат:** 4x рост оборота
- **Решение:** Перенос на Bitrix Магазин+CRM, интеграция с 1С, синхронизация товаров
- **Маркетинг:** 5+ маркетплейсов (Озон, WildBerries, Goods, Beru, Aliexpress), омниканальная модель

#### 1-й Конвейерный Завод
- **Результат:** 1400 руб/целевой звонок, 25% рост оборота и прибыли
- **Решение:** Внедрение Bitrix24 для 6 направлений (продажа, проектирование, закупка, бухгалтерия, производство, логистика)
- **Маркетинг:** Контекстная реклама, YouTube канал, ABM-маркетинг

## Установка и запуск

### Требования
- Node.js 22+
- pnpm 10+

### Локальная разработка

```bash
# Установка зависимостей
pnpm install

# Запуск dev сервера
pnpm dev

# Сайт будет доступен на http://localhost:3000
```

### Сборка для продакшена

```bash
# Сборка проекта
pnpm build

# Запуск продакшена
pnpm start
```
