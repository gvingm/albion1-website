# Восстановление дизайна сайта Альбион-1

## Проблема
Сайт отображается как "текстовый" без дизайна из-за отсутствия установленных зависимостей.

## Причина
Проект использует современный стек технологий:
- React 19 + TypeScript
- TailwindCSS 4 с OKLCH цветовой палитрой
- Vite как сборщик
- pnpm как менеджер пакетов

Без установленных зависимостей (`node_modules`) сайт не может загрузить CSS стили и компоненты.

## Решение

### Вариант 1: Полная установка зависимостей (рекомендуется)

1. **Установите Node.js v22.12.0 или выше**
   ```bash
   # Скачайте с https://nodejs.org/
   # Или используйте nvm/nvm-windows
   ```

2. **Установите pnpm**
   ```bash
   npm install -g pnpm
   ```

3. **Установите зависимости проекта**
   ```bash
   cd albion1-website
   pnpm install
   ```

4. **Запустите dev сервер**
   ```bash
   pnpm dev
   ```

5. **Соберите для продакшена**
   ```bash
   pnpm build
   pnpm start
   ```

### Вариант 2: Быстрое восстановление (без установки)

Используйте готовые HTML файлы с CDN TailwindCSS:

1. **working-homepage.html** - полностью рабочая главная страница с дизайном
2. **test.html** - простая версия для проверки

Эти файлы содержат весь дизайн и работают без установки зависимостей.

## Что было исправлено

1. **Исправлена ошибка в HTML** - в `client/index.html` была синтаксическая ошибка с двойным тегом `<title>`
2. **Созданы рабочие HTML версии** с CDN TailwindCSS для немедленного использования
3. **Проверен весь код** - структура проекта и компоненты корректны

## Структура проекта

```
albion1-website/
├── client/                    # Frontend React приложение
│   ├── src/
│   │   ├── components/        # React компоненты
│   │   ├── pages/            # Страницы
│   │   ├── contexts/         # Контексты React
│   │   ├── hooks/            # Кастомные хуки
│   │   ├── lib/              # Утилиты
│   │   └── index.css         # TailwindCSS стили
│   └── index.html            # HTML шаблон
├── server/                    # Express.js сервер
├── shared/                    # Общие типы и утилиты
└── package.json              # Зависимости проекта
```
